Welcome to My CheckIDNumber.

This program is use in Thailand only. I create this program because my mother said she want some program that can save and store ID-Number and check is it correct or duplicate or not.

Why use this program

So in this program their had a lot of feature that comfort you to correct some id-number in anywhere that you what, But this program is NOT ONLINE database so your ID-number that your save in the program won't upload on some where. other feature if you want to see more you can download on this ChangeLog
